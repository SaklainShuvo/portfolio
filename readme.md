## Featured Projects

### Project 1: GrubGo

Description: A full-stack food ordering and delivery platform that enables users to browse restaurants, place orders, and track deliveries in real-time.

Technologies:

* Backend: Python, Flask, SQLite, WTForms, SQLAlchemy ORM
* Frontend: HTML5, CSS3, JavaScript (ES6+), Tailwind CSS
* Authentication: Flask-Login with bcrypt password hashing
* API: RESTful endpoints with JSON responses

Links: [Repository](https://github.com/saklainshuvo/GrubGO)

## How to Use

1. Clone this repository:
      git clone https://github.com/saklainshuvo/portfolio.git
   

2. Open index.html in your web browser

3. Navigate through different sections:
   - Home: Introduction and hero section
   - About: Personal background and interests
   - Skills: Technical skills and competencies
   - Projects: Featured work and projects
   - Contact: Get in touch information
   - CV: Detailed curriculum vitae

## Customization

To customize this portfolio for your own use:

1. Update personal information in index.html and cv.html
2. Modify color scheme in index.css (CSS variables at the top)
3. Replace project details with your own projects
4. Update social media links and contact information
5. Add your own images or assets
